## Current Project

🔹 **[Quantum Tunneling Simulator](https://github.com/PeterTenholder/Quantum_Wave_Packet)** — A 1D wave-packet solver using the split-operator (FFT) method. Simulating transmission coefficients with goal to match energy-averaged theory to ~0.1%, with absorbing boundary layers and full parameter sweeps across barrier height and width.
